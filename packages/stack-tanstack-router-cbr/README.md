# 📝 ブログ管理システム (TanStack Router + Code-based Routing)

このパッケージは以下の技術スタックを使用しています：
- **ルーター**: TanStack Router
- **ルーティング方式**: Code-based Routing (コードベースルーティング)

## 特徴

- ルートの定義を手動でコード内に記述
- `createRoute()` 関数を使用してルートツリーを構築
- より柔軟な制御が可能だが、手動での管理が必要

## 開発

```bash
pnpm dev
```

## ビルド

```bash
pnpm build
```
